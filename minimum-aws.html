Pro Tip: Budget Alarms

Never trust "estimated" billing. Always set up a Billing Alarm in CloudWatch.

AWS Console > Billing > Budgets > Create Budget > Limit: $5.00

The "Zombie" Volume Trap

Terminating an EC2 instance does not automatically delete its storage volume unless you check the specific box during creation.

Always check the "Elastic Block Store" tab in the EC2 console after shutting down a project.
